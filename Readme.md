# Dev Hire

## About the Project

The project is an developer hiring platform that contains that contains the following pages / functionalities:

- An homepage which display a list of developers showing their names, asking price, avatar, banner and a like button that allows a developer to be added to favorites.

- A favorites page that allows a user see all the developers who they made favorites.

- A currency conversion button that allows a user to convert the asking price of a developer to a range of various currencies.

## Tools

The following tools were used for this project

- React JS (Using Vite)
- React Router
- Redux Toolkit
- Axios
- SASS / SCSS

The project is live on [Netlify](https://dev-hire-ui.netlify.app/)
